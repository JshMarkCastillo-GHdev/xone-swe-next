xone — Brand Kit (Concept 1: Hexagon Mark)
===========================================

FILES INCLUDED
--------------
SVG (vector, scalable — use for print & design tools):
  xone_logo.svg          — Full lockup, light version (transparent bg)
  xone_logo_inverse.svg  — Full lockup, dark/inverse version
  xone_icon.svg          — Hex mark only
  xone_wordmark.svg      — Wordmark only

PNG (raster — transparent background where applicable):
  xone_logo_2x.png       — Full lockup @2x (1040×320px)
  xone_logo_4x.png       — Full lockup @4x (2080×640px)
  xone_logo_inverse_2x.png
  xone_logo_inverse_4x.png
  xone_icon_2x.png       — Icon @2x (320×320px)
  xone_icon_4x.png       — Icon @4x (640×640px)
  xone_wordmark_2x.png
  xone_wordmark_4x.png

HOW TO OPEN IN PHOTOSHOP
-------------------------
  SVG → File > Open → select .svg → choose canvas size → rasterized layer
  PNG → File > Open → transparent-bg layer ready to use

BRAND COLORS
------------
  Navy (primary):  #1a1a2e
  Violet:          #533bda
  Cyan:            #22d3ee
  Light:           #f1f5f9
  White:           #ffffff

NOTE ON PSD FORMAT
------------------
PSD is a proprietary Adobe format that cannot be generated without
Photoshop. The SVG files are the vector equivalent — they can be
opened directly in Photoshop, Illustrator, Figma, Sketch, or Inkscape.
